# Nombre del Proyecto
Foes of the Green Kingdom

## Descripción
Eres un astronauta que llega a un reino alienígena llamado "Green Kingdom" y te han encomendado la tarea de explorar su reino para derrotar enemigos, a cambio de dejarte escapar del planeta y volver a casa 
y tu de paso aprobechas para investigar y recoger información de dicho reino alienigena.

Genero --> Plataformas en 2d
